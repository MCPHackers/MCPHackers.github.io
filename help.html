Welcome to the RetroMCP-Java wiki!

RetroMCP-Java is a Java rewrite of the Minecraft/Mod Coder Pack to provide support for many versions which were never supported by original MCP with full reobfuscation, including other assorted features.

RetroMCP-Java aims to be a more competent replacement than the original RetroMCP, aiming for ease of use, cross platformability, and debuggability when RMCP-Java fails inevitably fails.

  • Built-in Updater
  • Tiny V2-based Mappings
  • Supports Java 8+
  • Multi-threaded decompilation
  • Internet-based version repository
Have fun, modders. Now, it is yourcraft!
